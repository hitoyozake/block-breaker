# Block Breaker

## 概要
ブラウザで動作するシンプルなブロック崩しゲーム

## 技術
- HTML5 Canvas
- JavaScript (ES Modules)

## フェーズ
Phase1: パドルを左右に動かす
Phase2: ブロック配置
Phase3: ボールを動かす
Phase4: 当たり判定
Phase4.5: テスト環境の構築(DevContainer + Playwright)
Phase5: スコア
Phase6: 玉の残像（Trail）
Phase7: ブロックテクスチャ（block.png）
Phase8: パドルのビジュアル改善（角丸・中央ハイライト）
Phase9: マウス・タッチ操作対応

## 操作方法

### キーボード
| キー | 操作 |
|------|------|
| ←（左矢印） | パドルを左に移動 |
| →（右矢印） | パドルを右に移動 |
| スペース | ボールを射出 / リスタート |

### マウス・タッチ（スマホ対応）
| 操作 | 動作 |
|------|------|
| カーソル移動 / スワイプ | パドルが追随 |
| クリック / タップ | ボールを射出 / リスタート |

> タップ位置が画面左半分なら左斜め、右半分なら右斜め45度で発射

## ルール
- ボールがブロックに当たると消える
- ボールがパドルに当たると跳ね返る
- 画面下に落ちたらゲームオーバー

## 開発方針
- 1PR = 1機能
- 既存コードを壊さない
- 開発・テストは DevContainer で行う
